# 🏦 Simple Banking System (Python)

A beginner-friendly banking system built using Python.  
This project simulates basic banking operations like login, deposit, withdrawal, and account management.

---

## 🚀 Features

- User Registration
- Secure Login System
- Password Hashing (SHA-256)
- Deposit Money
- Withdraw Money
- Check Balance
- Delete Account

---

## 🧠 Concepts Used

- Python Functions
- Dictionaries
- Loops (while loops)
- Conditional Statements
- Input/Output
- Hashing (hashlib)

---

## 🔐 Security

Passwords are stored using SHA-256 hashing to ensure basic security.
